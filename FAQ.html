Frequently Asked Questions
For Fresher+
Help me to understand different roles of Software Industry?
i) Creative Designer/Graphic Designer: He is responsibile to design the look and feel of any software (Mobile app/Website), creating Logos, image icons, etc. He creates the design using tools like Photoshop, Illustrator, CorelDraw, etc.
Main Skill set required : Creativity [No coding background required]
ii) Software Developer: His main responsbility is to develop the software (by logic and code) Main Skill set required: Passion for Coding (irrespective of any language/platform)
iii) Quality Assurance: Main responsibilty is to assure the Software developed is of good quality and useful for the users. He tests the Software either by Manual testing or by Automation.
Main Skill set required: Keen observation skills [Basics of Coding]

What certificate will I get after 6 months ?
No certificate. After 6 months, you will get the experience letter and that will be accepted and valued through out the Industry.
Is English communication (speak and understand) must for to join it?
